การหาค่า BMR

  • ค่าคงที่ผู้ชาย = + 5
  • ค่าคงที่ผู้หญิง = - 161

  • ผู้ชาย: BMR = (10 x น้ำหนัก กก.) + (6.25 x ส่วนสูง ซม.) - (5 x อายุ) + 5
  • ผู้หญิง: BMR = (10 x น้ำหนัก กก.) + (6.25 x ส่วนสูง ซม.) - (5 x อายุ) - 161

    • ช่วยให้คุณรู้ พลังงานขั้นต่ำที่ร่างกายต้องการต่อวัน เพื่อการทำงานพื้นฐาน (หายใจ, สูบฉีดเลือด) ทำให้วางแผน ควบคุมน้ำหนัก ได้อย่างมีประสิทธิภาพ
